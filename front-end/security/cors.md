## 关闭cors

打开 chrome://flags/， CORS for content scripts 关闭，还可以使用扩展插件 “moesif CORS extension”， 火狐也可以使用插件 "CORS Everywhere"。

关闭samesite校验, 打开 chrome://flags/, SameSite by default cookies 配置 disabled
